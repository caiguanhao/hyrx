hyrx
====

Starts a local server serving modified content from hengyirong.com.
Slow images and scripts are removed and some handy scripts are added.

Needs `golang:1.7.1`

```
go get -v github.com/caiguanhao/hyrx
```

To build for Windows EXE:

Run `go build` with these environment variables: `GOOS=windows GOARCH=386`
